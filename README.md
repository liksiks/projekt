## Название варианта
Movie Library (Личная кинотека)

## Краткое описание программы

**«Movie Library»** — десктоп‑приложение с графическим интерфейсом (GUI) для учёта и систематизации личной коллекции фильмов.

#### Основные возможности:

1.  **Добавление фильмов** через удобную форму с полями:
    
    -   название;
        
    -   жанр;
        
    -   год выпуска;
        
    -   рейтинг (по 10‑балльной шкале).
        
2.  **Визуализация данных** в виде интерактивной таблицы с возможностью просматривать всю коллекцию в структурированном виде.
    
3.  **Фильтрация контента** по двум критериям:
    
    -   по жанру (выпадающий список с доступными жанрами);
        
    -   по году выпуска (выпадающий список с доступными годами).  
        Также предусмотрена кнопка сброса фильтров для возврата к полному списку.
        
4.  **Валидация вводимых данных**:
    
    -   проверка заполнения обязательных полей;
        
    -   контроль формата года (только число в разумном диапазоне: 1888–2030);
        
    -   ограничение рейтинга (от 0 до 10).
        
5.  **Сохранение и загрузка данных** в формате JSON:
    
    -   все добавленные фильмы автоматически сохраняются в файл `movies.json`;
        
    -   при запуске приложение загружает ранее сохранённые данные;
        
    -   обеспечивается целостность коллекции между сеансами работы.
        
6.  **Контроль версий через Git**:
    
    -   проект организован как Git‑репозиторий;
        
    -   включён файл `.gitignore` для исключения временных и служебных файлов из отслеживания;
        
    -   история изменений фиксируется стандартными средствами Git.
        
7.  **Документация**:
    
    -   прилагается файл `README` (в формате Markdown) с описанием проекта, авторством, инструкцией по использованию и примерами тестирования.
## Инструкция по запуску (что установить, как запустить)

#### Шаг 1. Установка необходимых компонентов

1.  **Установите Python версии 3.6 или выше**:
    
    -   Перейдите на официальный сайт Python: python.org.
        
    -   Скачайте установщик для вашей операционной системы (Windows, macOS или Linux).
        
    -   Во время установки **обязательно отметьте галочку «Add Python to PATH»** (для Windows).
        
    -   Завершите установку, следуя инструкциям на экране.
        
2.  **Проверьте установку Python**:
    
    -   Откройте командную строку (Windows: `Win + R` → введите `cmd` → Enter) или терминал (macOS/Linux).
        
    -   Введите команду:
        
        ```
        python --version
        
        ```
        
    -   Если отображается версия Python (например, `Python 3.11.5`), установка прошла успешно.
        

#### Шаг 2. Получение кода проекта

Выберите один из способов:

**Способ 1. Клонирование репозитория Git** (если у вас установлен Git):

1.  Откройте терминал или командную строку в папке, куда хотите сохранить проект.
    
2.  Выполните команду:
    
    ```
    git clone <URL_репозитория>
    
    ```
    
3.  Перейдите в папку проекта:
    
    ```
    cd Movie_Library
    
    ```
    

**Способ 2. Скачивание ZIP‑архива**:

1.  На странице репозитория нажмите кнопку **«Code»** → **«Download ZIP»**.
    
2.  Распакуйте архив в удобное место на вашем компьютере.
    
3.  Откройте распакованную папку.
    

#### Шаг 3. Запуск приложения

1.  Убедитесь, что вы находитесь в папке проекта (где лежит файл `main.py`).
    
2.  Запустите приложение командой:
    
    ```
    python main.py
    
    ```
    
    -   Для Linux/macOS может потребоваться команда:
        
        ```
        python3 main.py
        
        ```
        
3.  Через несколько секунд откроется окно приложения «Movie Library».
    

#### Шаг 4. Проверка работоспособности

Проведите краткий тест:

1.  **Добавьте фильм**:
    
    -   В поле «Название» введите: `Начало`.
        
    -   В поле «Жанр» введите: `Фантастика`.
        
    -   В поле «Год выпуска» введите: `2010`.
        
    -   В поле «Рейтинг» введите: `8.8`.
        
    -   Нажмите кнопку **«Добавить фильм»**.
        
    -   Убедитесь, что фильм появился в таблице.
        
2.  **Протестируйте фильтрацию**:
    
    -   Выберите в фильтре по жанру: `Фантастика`.
        
    -   Проверьте, что отображаются только фильмы этого жанра.
        
    -   Нажмите **«Сбросить фильтры»** — должен появиться полный список.
        
3.  **Проверьте сохранение данных**:
    
    -   Закройте приложение (крестик в углу окна).
        
    -   Снова запустите его командой `python main.py`.
        
    -   Убедитесь, что добавленный фильм остался в списке.
        

----------

### Возможные проблемы и их решения

-   **Ошибка: «python: command not found»**
    
    -   Убедитесь, что Python установлен и добавлен в PATH. Переустановите Python, отметив галочку «Add to PATH».
        
-   **Ошибка импорта tkinter**
    
    -   `tkinter` входит в стандартную поставку Python. Если ошибка возникает, переустановите Python — выберите полную установку (включая Tcl/Tk).
        
-   **Файл `movies.json` не создаётся**
    
    -   Проверьте права доступа к папке проекта. Запустите терминал/командную строку от имени администратора.
        
-   **Приложение запускается, но таблица пуста**
    
    -   Это нормально при первом запуске. Добавьте фильм через форму — файл `movies.json` создастся автоматически.
        
-   **Фильтрация не работает**
    
    -   Убедитесь, что в выпадающих списках фильтров выбраны значения, отличные от «Все».

## Примеры использования 
#### Пример 1. Добавление фильма

**Цель:** внести новый фильм в коллекцию.

**Шаги:**

1.  Запустите приложение (команда `python main.py`).
    
2.  Заполните поля формы:
    
    -   **«Название»**: `Крёстный отец`;
        
    -   **«Жанр»**: `Драма`;
        
    -   **«Год выпуска»**: `1972`;
        
    -   **«Рейтинг»**: `9.2`.
        
3.  Нажмите кнопку **«Добавить фильм»**.
    

**Результат:**

-   фильм появляется в таблице ниже формы;
    
-   все поля ввода очищаются для следующего добавления;
    
-   данные сохраняются в файл `movies.json`.
    

----------

#### Пример 2. Добавление нескольких фильмов

**Цель:** заполнить коллекцию несколькими фильмами подряд.

**Шаги:**

1.  Добавьте первый фильм:
    
    -   **Название:**  `Интерстеллар`;
        
    -   **Жанр:**  `Фантастика`;
        
    -   **Год выпуска:**  `2014`;
        
    -   **Рейтинг:**  `8.6`.
        
2.  Добавьте второй фильм:
    
    -   **Название:**  `Начало`;
        
    -   **Жанр:**  `Фантастика`;
        
    -   **Год выпуска:**  `2010`;
        
    -   **Рейтинг:**  `8.8`.
        
3.  Добавьте третий фильм:
    
    -   **Название:**  `Побег из Шоушенка`;
        
    -   **Жанр:**  `Драма`;
        
    -   **Год выпуска:**  `1994`;
        
    -   **Рейтинг:**  `9.3`.
        

**Результат:**

-   в таблице отображаются все три фильма;
    
-   в фильтрах по жанру появились варианты: `Фантастика`, `Драма`;
    
-   в фильтре по году — `1994`, `2010`, `2014`.
    

----------

#### Пример 3. Фильтрация по жанру

**Цель:** отобразить только фильмы определённого жанра.

**Шаги:**

1.  В выпадающем списке **«Жанр»** фильтра выберите `Фантастика`.
    
2.  Нажмите (или дождитесь автоматического применения фильтра).
    

**Результат:**

-   в таблице остались только фильмы жанра `Фантастика` (`Интерстеллар` и `Начало`);
    
-   остальные записи скрыты.
    

----------

#### Пример 4. Фильтрация по году выпуска

**Цель:** найти фильмы, выпущенные в конкретном году.

**Шаги:**

1.  В выпадающем списке **«Год выпуска»** фильтра выберите `1994`.
    
2.  Примените фильтр.
    

**Результат:**

-   в таблице отображается только фильм `Побег из Шоушенка` (выпущен в 1994 году);
    
-   фильмы других годов скрыты.
    

----------

#### Пример 5. Сброс фильтров

**Цель:** вернуть полный список фильмов после фильтрации.

**Шаги:**

1.  Убедитесь, что какой‑либо фильтр активен (например, показан только один фильм).
    
2.  Нажмите кнопку **«Сбросить фильтры»**.
    

**Результат:**

-   таблица обновляется и показывает все добавленные фильмы;
    
-   оба фильтра устанавливаются в значение `Все`.
    

----------

#### Пример 6. Проверка сохранения данных

**Цель:** убедиться, что коллекция сохраняется между сеансами работы.

**Шаги:**

1.  Добавьте один или несколько фильмов (как в примере 1 или 2).
    
2.  Закройте приложение (крестиком в углу окна).
    
3.  Снова запустите приложение командой `python main.py`.
    

**Результат:**

-   приложение загружает данные из файла `movies.json`;
    
-   все ранее добавленные фильмы отображаются в таблице;
    
-   фильтры обновляются с учётом текущей коллекции.
    

----------

#### Пример 7. Проверка валидации данных

**Цель:** протестировать защиту от некорректного ввода.

**Сценарий 1. Некорректный год:**

1.  Введите в поле **«Год выпуска»** текст `две тысячи двадцать`.
    
2.  Заполните остальные поля и нажмите **«Добавить фильм»**.
    
3.  **Результат:** появляется сообщение об ошибке «Год должен быть числом», фильм не добавляется.
    

**Сценарий 2. Рейтинг вне диапазона:**

1.  Введите в поле **«Рейтинг»** значение `15`.
    
2.  Заполните остальные поля и нажмите **«Добавить фильм»**.
    
3.  **Результат:** выводится ошибка «Рейтинг должен быть от 0 до 10», запись не создаётся.
    

**Сценарий 3. Пустое поле:**

1.  Оставьте поле **«Название»** пустым.
    
2.  Заполните остальные поля и нажмите **«Добавить фильм»**.
    
3.  **Результат:** показывается ошибка «Введите название фильма», добавление отменяется.
